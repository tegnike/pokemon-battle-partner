# クチート

## 基本メモ
- 対象: Pokemon Champions シングル Season M-3 使用率上位25位。
- ローカルID: mawile
- 統計キー: クチート
- タイプ: はがね/フェアリー
- 種族値: HP50 / 攻撃85 / 防御85 / 特攻55 / 特防55 / 素早50
- 役割概要: メガクチートの高火力じゃれつく、ふいうち、つるぎのまいで低速高圧力を出す。
- まず疑う型: クチートナイトを持ち、じゃれつくを中心に動く型。

## 対戦での見方
- 技はじゃれつく (98.70%)、ふいうち (98.60%)、つるぎのまい (87.70%)、かみなりパンチ (30.40%)が目立つ。
- 持ち物はクチートナイト (99.60%)、きあいのタスキ (0.10%)、いのちのたま (0.00%)、オボンのみ (0.00%)が目立つ。
- 性格はいじっぱり (84.80%)、ゆうかん (12.90%)、ようき (0.70%)、わんぱく (0.50%)が目立つ。
- 能力ポイント配分は32-32-2-0-0-0 (27.40%)、32-32-0-0-2-0 (16.20%)、32-32-0-0-0-2 (7.60%)、32-32-1-0-1-0 (6.50%)が目立つ。
- 同居ポケモンはミミッキュ、ガブリアス、カバルドン、ギャラドスが目立つ。
- 先発で出た場合は、じゃれつくだけでなく補助技・交代技・積み技の有無を確認する。
- 裏から出た場合は、削れた相手をじゃれつくや先制技で縛る動きか、クッションとして受けに来た動きかを分ける。
- クチートナイトが見えた時点で、メガシンカ前後、こだわり固定、タスキ残り、回復込みの耐久を再評価する。
- ミミッキュと並んでいる時は、初手展開からクチートを通すのか、クチートで荒らして裏を通すのかを見極める。
- 相手の技を1つ見ただけで型を固定しない。統計上は複数の役割が混在している。

## こちらの判断メモ
- 持ち物が判明するまで、タスキ・スカーフ・メガストーンのどれかを常に候補に残す。
- 同居率の高いポケモンまで含めて、先発展開か終盤エースかを見分ける。
- こちらのガブリアス、アシレーヌ、メタグロス、ウォッシュロトム、マスカーニャ、サザンドラの誰で処理するかを、持ち物判明後に更新する。
- 一撃で倒せる場面でも、タスキ、ばけのかわ、先制技、メガシンカ後の耐久変化、天候や壁を必ず確認する。
- 交代読みよりも、相手の一番自然な勝ち筋を止める手を優先する。
